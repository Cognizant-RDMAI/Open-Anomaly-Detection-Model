# Contributing to BB3AB

Thank you for considering contributing to BB3AB! We welcome feedback and suggestions, but please note that **code contributions (such as pull requests, forks, or edits) are not permitted at this time**.
This repository is part of an internal research and development initiative and is intended to showcase the project outcomes to the wider community.

## How to Contribute

Currently, there is no Contributor License Agreement provided for this project. The team will consider comments and suggestions and incorporate as and when feasible and value adding.

## Reporting Issues

If you encounter a bug or have a feature request, please contact the Cognizant partner to report an issue. When reporting an issue, please include:

- A clear and descriptive title.
- A detailed description of the problem or suggestion.
- Steps to reproduce the issue (if applicable).
- Any relevant logs, screenshots, or error messages.
- Suggesting Enhancements

We welcome suggestions for new features or improvements. To suggest an enhancement:

- A clear and descriptive title.
- A detailed explanation of the enhancement.
- Any benefits or use cases for the enhancement.
